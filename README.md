# Goals Widget

Небольшой Windows-виджет на Electron для финансовой цели, капитала и задач из Markdown-файлов. Виджет хранит задачи в обычных `.md` файлах, поэтому их удобно открывать и синхронизировать через Obsidian, Git или любой другой редактор.

## Что умеет

- показывает финансовую цель, дедлайн, прогресс и нужные взносы;
- учитывает личный капитал, капитал в управлении и годовую доходность;
- добавляет доход к личному капиталу;
- читает задачи из Markdown-файлов формата `- [ ] Задача`;
- поддерживает подзадачи, редактирование, удаление, приоритет и перенос задач между файлами;
- запоминает выбранный файл задач и размеры окна.

## Требования

- Windows 10 или Windows 11;
- Node.js LTS: https://nodejs.org/
- Git: https://git-scm.com/

Проверить установку можно в PowerShell:

```powershell
node -v
npm -v
git --version
```

## Быстрый запуск

1. Скачайте или клонируйте репозиторий.

```powershell
git clone https://github.com/YOUR_NAME/Goals_Widget.git
cd Goals_Widget
```

2. Установите зависимости.

```powershell
npm install
```

3. Запустите виджет.

```powershell
npm start
```

## Где лежат задачи

По умолчанию приложение работает с папкой:

```text
C:\Users\HomePC\Documents\Obsidian\Progects\MyLife
```

Если вы запускаете виджет на другом компьютере, измените путь в [main.js](main.js):

```js
const TARGET_DIR = "C:\\Users\\HomePC\\Documents\\Obsidian\\Progects\\MyLife";
```

Например:

```js
const TARGET_DIR = "C:\\Users\\YourUser\\Documents\\Obsidian\\MyLife";
```

При первом запуске виджет сам создаст папку и файл `Дела.md`, если их еще нет.

## Формат задач

Обычная задача:

```markdown
- [ ] Сделать платеж
```

Выполненная задача:

```markdown
- [x] Сделать платеж
```

Подзадача:

```markdown
- [ ] Большая задача
    - [ ] Маленький шаг
```

## Автозапуск через VBS

В проекте есть [TargetWiget.vbs](TargetWiget.vbs). Его можно использовать как тихий запуск без открытого окна консоли. Если путь к проекту изменился, откройте файл и проверьте, что он указывает на правильную папку.

Для автозапуска Windows:

1. Нажмите `Win + R`.
2. Введите `shell:startup`.
3. Поместите ярлык на `TargetWiget.vbs` в открывшуюся папку.

## Локальные файлы

Эти файлы создаются автоматически и не отправляются в Git:

- `node_modules/` - установленные зависимости;
- `config.json` - положение и размер окна;
- `log.txt` - журнал действий;
- `.env` - локальные переменные, если появятся.

## Если что-то не запускается

1. Убедитесь, что Node.js установлен: `node -v`.
2. Выполните `npm install`.
3. Проверьте путь `TARGET_DIR` в `main.js`.
4. Запустите из PowerShell командой `npm start`, чтобы увидеть ошибку.
